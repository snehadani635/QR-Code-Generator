# 📱 QR Code Generator

## 📸 Project Preview

![QR Code Generator](screenshot.qr.png)

A simple and responsive QR Code Generator built using HTML, CSS, and JavaScript. This application allows users to generate QR codes instantly from any text or URL using a free QR Code API.

## Features

- Generate QR codes from text or URLs
- Instant QR code generation
- Clean and responsive UI
- Input validation
- Error animation for empty input
- Smooth QR code display animation

## Technologies Used

- HTML5
- CSS3
- JavaScript
- QR Server API

## How It Works

1. Enter any text or URL in the input field.
2. Click the **Generate QR Code** button.
3. The application sends the entered data to the QR Server API.
4. A QR code image is generated and displayed instantly.
5. If the input field is empty, an error animation is shown.
